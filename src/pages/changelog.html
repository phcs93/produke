Alpha 0.3

October ??, 2025 ?? de Outubro de 2025

The focus of this update was to make DMOs functional again. It packs a lot of bug fixes and new features for demo playback. O foco dessa atualização foi fazer as DMOs ficarem funcionais novamente. Ela trás várias correções de bugs e novos recursos para visualização de demos. A very hard work was made to also fix playback of original demos from DOS (Duke Nukem 3D 1.5 Atomic Edition). This is broken in xDuke and all ports that derive from it. Foi feito um arduo trabalho para também corrigir a reprodução das demos originais de DOS (Duke Nukem 3D 1.5 Atomic Edition). Isso está quebrado no xDuke e em todos os ports que derivem dele. Another very important feature in this version is the IPv6 native support (full dual-stack). Outra funcionalidade muito importante nessa versão é o suporte nativo à IPv6 (full dual-stack).

New Features Novas Funcionalidades

  • Native IPv6 support. Suporte IPv6 nativo.
  • Automatic screenshots. Screenshots automáticas.
  • Demo autorecord. Gravação automática de demos.
  • Chat saved inside DMO file. Chat salvo dentro do arquivo DMO.
  • Map CRC saved inside demo file to prevent usermap name conflicts. CRC do mapa salvo dentro do arquivo demo para previnir conflitos com nomes de usermaps.
  • Date and time of match saved inside of DMO file. Data e hora da partida salvas dentro da DMO.
  • Open DMO files with proDuke using double click or drag and drop. Abrir DMO com dois cliques ou arrastando e soltando.
  • Option to display outlines and overhead info while watching DMOs (players, projectiles, markers, holodukes and dropped weapons). Opção para exibir contornos e informações de jogadores ao assistir DMOs (jogadores, projéteis, marcadores, holodukes e armas derrubadas).
  • Damage numbers (new netflag). Números de dano (nova netflag).
  • Various other new netflags. Várias outras netflags novas.
  • Support for custom crosshair tiles via crosshair.art. Suporte à tiles customizados de mira com a crosshair.art.
  • New crosshair option to indicate current autoaim mode (A.A. Tied). Nova opção de mira para indicar o modo de autoaim atual (A.A. Tied).
  • New Weapon Auto Switch option based on the weapon preference list. Novo modo de trocade arma automatica baseado na lista de preferencia de armas.
  • New scorebar modes. Novos modos de barra de pontuação.
  • Shows a spinning nuke icon above the player's head when he's tying or on the menus. Mostra um ícone do duke girando acima da cabeça de um player que estiver digitando ou com o menu aberto.
  • Message hotkeys and chat behavior options. Atalhos de mensagem e opções de comportamento do chat.
  • Offline multiplayer menu. Menu de multiplayer offline.
  • Air display on HUD. Exibição de ar no HUD.
  • New obituaries mode (hybrid). Novo modo de obituários (híbrido).
  • Show player who pressed the nuke button. Aviso de quem apertou o nuke button.
  • Secrets and monsters kill count while displaying the map. Contagem de secrets e de monstros mortos ao exibir o mapa.

Improvements Melhorias

  • Static SDL.DLL and SDL_MIXER.DLL embeded inside the produke.exe. No need to have these files in the game directory anymore. This also improves game performance. SDL.DLL e SDL_MIXER.DLL estáticas incorporadas dentro do produke.exe. Não é mais necessário ter esses arquivos no diretório do jogo. Isso também aumenta a performance do jogo.
  • Now the game reads it's mandatory files from produke.grp. No need to have LOOKUP.DAT and TILES023.ART in the game directory anymore. Agora o jogo lê os arquivos obrigatórios direto do produke.grp. Não é mais necessário ter o LOOKUP.DAT e o TILES023.ART no diretório do jogo.
  • Screenshots are now saved as 8bit PNG files (smaller disk size). As screenshots agora são salvas no formato PNG de 8 bits (tamanho menor em disco).
  • Now demos keep recording even after a player quits. Agora a demo continua sendo gravada mesmo depois que alguém quitar.
  • Now the game records a new demo when changing maps. Agora o jogo começa a gravar uma demo nova quando acontece uma troca de mapa.
  • Demos now record current items and weapons of each player when advancing levels in COOP modes. Agora as demos gravam os items e armas atuais de cada jogador ao avançar de mapa em modos COOP.
  • Now it's possible to play with up to 15 BOTs. (16 players in total). Agora é possível jogar com até 15 BOTs (16 jogadores no total).
  • Yellow latency indicador changed from 5 game tics (~166ms) to 3 games tics (~100ms). Indicador de latencia amarelo alterado de 5 game tics (~166ms) para 3 games tics (~100ms).
  • Obituaries now detect more death types, like monsters and environment damage. Os obituários agora exibem mais tipos de morte, como monstros e dano do ambiente..
  • Now the game shows who paused the game. Agora o jogo mostra quem pausou o jogo.
  • New netflag presets. Novos presets de netflags.
  • Improved weapon preference menu selection. Melhoria na seleção do menu de preferencia de armas.
  • Smaller font in submenus (Duke 1.3 font art). Fonte menor nos submenus (Arte do Duke 1.3).
  • Purple palette in outlines now show as propper purple instead of orangeish. A paleta roxa nas outlines agora aparece corretamente como roxo ao invés de alaranjada.
  • The game menu won't open up anymore when exiting the console by pressing ESC. O menu do jogo não será mais aberto quando você estiver saindo do console com a tecla ESC.
  • Now you can only close the console by pressing ESC. This is done so you can type the configured console key in the terminal if you want. Also, the configured key will not be typed automaticaly into the terminal if you're just opening the console. Agora só é possivel fechar o console apertando ESC. Isso foi feito para que você consiga digitar o caractere configurado para abrir o console no terminal. Além disso, a tecla configurada não será mais digitada automaticamente no terminal se você estiver apenas abrindo o console.
  • Console now supports shifted keys (!, @, #, etc) Agora o console suporta teclas com shift (!, @, #, etc).

Bug Fixes Correções de bugs

  • Fixed DMO files not being recorded correctly. Correção das DMOs não estarem sendo gravadas corretamente.
  • Fixed playback of xDuke DMOs. Correção do playback de DMOs do xDuke.
  • Fixed a bug where an empty message is sent if you press any key but Enter. Correção de um bug que fazia uma mensagem vazia ser enviada ao não apertar Enter.
  • Fixed a bug where an incorrect sprite was shown in overhead info if user nickname contained spaces. Correção de um bug onde uma sprite incorreta aparecia no overhead info caso o nome do jogador contivesse espaços.
  • Fixed a bug where the game would desync while playing Terminator mode with duke talk disabled. Correção de um bug que causava out of sync quando se estava jogando Terminator com o duke talk desligado.
  • Fixed a bug where weapon preferences was not being saved to CFG. Correção de um bug onde as preferencias de armas não estavam sendo salvas no CFG.
  • Fixed a bug where allied weapons would show through walls even with team outlines disabled. Correção de um bug onde a arma dos aliados aparecia pela parede mesmo com os contornos de time desligados.
  • Fixed a bug where default match netflags were not being set during game initialization. Correção de um bug onde as netflags padrão não estavam sendo consideradas durante a inicialização do jogo.
  • Fixed a bug where player shadows were being rendered in SoS (sector over sector) situations. Correção de um bug onde as sombras dos jogadores estavam sendo renderizadas em situações de SoS (sector over sector).
  • Fixed a bug where outlines and overhead info sprites were being affected by the map's global visiblity. Correção de um bug onde os contornos e o overhead info estavam sendo afetados pela visiblidade global do mapa.
  • Fixed a bug where the game would close if opened without a duke3d.cfg. Correção de um bug onde o jogo fechava ao abrir sem um duke3d.cfg.
  • Fixed a bug where the player color would show incorrectly in the menu. Correção de um bug onde a cor do jogador aparecia incorreta no menu.
  • Fixed a bug where some sprites would clip through the walls when widescreen is enabled. This is a bug present in rDuke's code. Correção de um bug onde algumas sprites cortavam pelas paredes quando o modo widescreen estava habilitado. Isso é um bug presente no código do rDuke.

Original Bug Fixes (from xDuke) Correções de bugs originais (do xDuke)

  • Fixed playback of original DOS DMOs (demos 1, 2 and 3 from Duke Nukemm 3D 1.5 Atomic GRP). Correção do playback de DOS DMOs originais (demos 1, 2 e 3 da GRP do Duke Nukem 3D 1.5 Atomic).
  • Fixed a bug in which the game wouldn't properly release the ALT key (often when doing ALT+TAB), so it would consider it to be pressed while it wasn't, potentially resulting in FullScreen toggling when the user pressed ENTER. Correção de um bug no qual o jogo falhava ao soltar a tecla ALT (comum ao dar ALT+TAB), o que resultava em considerá-la como pressionada quando na verdade não estava, o que potencialmente resultava em (des)abilitar a tela cheia quando o usuário apertasse ENTER.
  • Demo cams work again (it was disabled in xDuke) and can be toggled on and off. Cameras de demo funcionam novamente (estavam desativadas no xDuke) e podem ser ligadas ou desligadas.
  • Fixed the famous E4L11 (Area 51) crash that happens when a player goes through the air shaft in the outside area. Correção do famoso crash da E4L11 (Area 51) que ocorre quando um jogador passa pelo duto de ar da parte de fora do mapa.
  • Fixed a crash that could happen in E1L2 (Red Light District) when viewing the automap near the dark corridor. Correção de um crash que ocorria na E1L2 (Red Light District) quando se abria o automap perto do corredor escuro.
  • Fixed a bug where global palettes (nightvision/underwater) would reset when switching from fullscreen to windowed and vice-versa. Correção de um bug onde paletas globais (nightvision/em baixo d'água) eram resetadas ao trocar do modo fullscreen para em janela ou vice-versa.

Original Bug Fixes (from DOS) Correções de bugs originais (do DOS)

  • Fixed a bug where the player would respawn shrunk in some situations. Correção de um bug onde o jogador renascia encolhido em algumas situações.
  • Fixed a bug where the crane would pick the closest player when the grabbed player dies. Correção de um bug onde o guindaste pegava o jogador mais próximo quando o jogador agarrado morria.
