Nightvision will ignore all sectors visibility. In the original game, all it does is set a different global palette as well as set the palette of certain sprites. It's recommended to keep 
Netplay Options > NetFlags > [DOS] Useless Nightvision
toggled off for better results (i.e. player glow set to minimum so it~s nto affected by sector shade). O Nightvision vai ignorar a visibilidade de todos os setores. No jogo original, ela apenas troca a paleta global além de setar a paleta de certas sprites. É recomendado deixar a opção 
Netplay Options > NetFlags > [DOS] Useless Nightvision
desligada para melhores resultados (i.e. brilho dos players setados par ao valor mínimo para que não seja afetado pelo shading do setor).
