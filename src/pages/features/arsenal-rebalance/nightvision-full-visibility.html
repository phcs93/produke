Nightvision will ignore all sectors visibility and make everything fully visible and set shade to minimal. In the original game, all it does is set a different global palette as well as set the palette of certain sprites. A nightvision vai ignorar a visibilidade de todos os setores e deixar tudo completamente visivel e com o shade no mínimo. No jogo original, ela apenas troca a paleta global além de setar a paleta de certas sprites.
