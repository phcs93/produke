There is a new scorebar type to free up more screen space. Existe um novo tipo de barra de pontuação para liberar mais espaço na tela. There are a few options you can choose: Existem algumas opções que você pode escolher:
TYPE TIPO DESCRIPTION DESCRIÇÃO
OFF Hides the scorebar entirely. Esconde a barra de pontuação completamente.
CLASSIC At the top like the original game. Exibe a barra no topo assim como no jogo original.
MODERN At the top right corner, without taking up screen space. No canto superior direito, sem ocupar espaço da tela.
