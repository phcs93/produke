Now, under 
Netplay Options
, there is a new menu called 
NetFlags
. You can toggle on/off most of the new features through here. Also, some of the original game options (shown in blue) have been moved to here, like setting wheter the match will be "spawn" or "no spawn", coop friendly fire, starting with keycards or not, etc. Agora, dentro do menu 
Netplay Options
, existe um novo menu chamado 
NetFlags
. Você pode ligar/desligar a maioria das novas funcionalidades por aqui. E também algumas das opções originais do jogo (exibidas em azul) foram movidas para cá, como configurar se a partida vai ser "spawn" ou "no spawn", fogo amigo no coop, começar a partida com os keycards ou não, etc. The number shown above the menu is a bitfield representing all enabled flags, go to command line arguments to learn more. You can use the args builder to generate the argument. O número exibido acima do menu é um bitfield que representa todas as flags habilitadas, vá para argumentos de linha de comando para entender mais. Você pode utilizar o construtor de argumentos para gerar o argumento.

Faz com que você sempre renasça no ponto de renascimento mais distante em relação ao local de ondê você morreu, ao invés de pegar aleatoriamente um dos pontos disponíveis. Essa opção deve permanecer desativada para que os modos CTF funcionem corretamente.

Esta opção simula o modo CTF do Shadow Warrior.

Durante o gameover (i.e. sem vidas extras para renascer), não será possível espiar (com o spycam/coop view) os jogadores aliados.
