You can now load an user map directly from the game's menu. After "Episode 4" there will be a new option called "User Map". Você pode carregar um user map diretamente pelo menu do jogo. Depois do "Episode 4" você verá uma nova opção chamada "User Map". Clicking on this new option will show a list of all maps in your configured user maps folder. You can press a character key and the cursor will be positioned at the first entry that begins with that letter. Clicar nessa nova opção vai exibir uma lista com todos os user maps que estiverem no diretorio de user maps configurado. Apertar uma tecla fará com que o cursor seja posicionado no primeiro mapa que comece com a letra pressionada. To configure your user maps directory, just edit your 
duke3d.cfg
file. Under the 
[Misc]
section there is a new entry called 
MAPDir
. You can set a full path or a relative path. It defaults to a "maps/" in the root directory of the game executable. Para configurar o diretório de user maps, edite o seu arquivo 
duke3d.cfg
