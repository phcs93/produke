A player will no longer respawn already drowning if player was drowning when he last died. This can be disabled to match the original DOS behavior at the menu 
Netplay Options > NetFlags > [DOS] Can Respawn Drowning
. O jogador não vai mais renascer já se afogando caso tenha morrido por afogamento anteriormente. Isso pode ser desabilitado para ficar como era no jogo original pelo menu 
Netplay Options > NetFlags > [DOS] Can Respawn Drowning
