Fixed bugs related to map start on COOP matches:
  • (a) Everyone would start the next map with P1's health points;
  • (b) When passing to the next map while holding a weapon with no ammo, the player start holding that weapon and the game will NOT switch automatically to another weapon with ammo;
These can be restored to match the original DOS behavior at the menu 
Netplay Options > NetFlags > [DOS] Map Start Bugs
. Correção de bugs relacionados ao início de mapa ao jogar o modo COOP:
  • (a) Todo mundo iniciará o próximo mapa com os pontos de vida do P1;
  • (b) Ao passar para o proximo mapa segurando uma arma sem munição, o player começará segurando essa mesma arma e o jogo NÃO trocará sozinho para outra arma que tenha munição;
Essas correções podem ser desabilitadas para ficar como era no jogo original pelo menu 
Netplay Options > NetFlags > [DOS] Map Start Bugs
