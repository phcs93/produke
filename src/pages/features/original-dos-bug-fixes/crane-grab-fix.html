Fixed a bug where the crane would pick the closest player when the grabbed player dies. This can be disabled to match the original DOS behavior at the menu Netplay Options > NetFlags > [DOS] Crane Dead Player Swap. Correção de um bug onde o guindaste pegava o jogador mais próximo quando o jogador agarrado morria. Isso pode ser desabilitado para ficar como era no jogo original pelo menu 
Netplay Options > NetFlags > [DOS] Crane Dead Player Swap
