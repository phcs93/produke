Fixed a bug where the game would default to P1's global palette (e.g. underwater palette) when performing screen flashes (e.g. when picking up an item) when displaying other player's screen when watching demos OR when peeking and ally's screen. Correção de um bug onde o jogo usava por padrão a paleta global (e.g. paleta de embaixo d'agua) do P1 ao fazer as piscadas de tela (e.g. quando se pega um item) enquanto mostrava a tela de outros players durante a reprodução de demos OU ao ver a tela de um aliado.
