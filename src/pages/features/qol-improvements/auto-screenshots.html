You can configure the game to take screenshots automatically based on some triggers through the menu 
Options > Recording Setup > Auto Screenshots
. Você pode configurar o jogo para tirar screenshots automaticamente baseado em alguns gatilhos através do menu 
Options > Recording Setup > Auto Screenshots
. The screenshot will be saved in the screenshots folder with the corresponding trigger suffix. A screenshot será salva na pasta de screenshots com o sufixo correspondete ao gatilho.
TRIGGER GATILHO DESCRIPTION DESCRIÇÃO
LIMIT HIT Takes a screenshot when the score limit is hit. Tira uma screenshot quando o limite de pontuação é atingido.
QUIT Takes a screenshot when a player quits from the match. Tira uma screenshot quando um jogador quita da partida.
NUKEBUTTON Takes a screenshot when a player hits the nuke button. Tira uma screenshot quando um jogador aperta o nuke button.
PAUSE Takes a screenshot when a player pauses the game. Tira uma screenshot quando um jogador pausar o jogo.
UNPAUSE Takes a screenshot when a player unpauses the game. Tira uma screenshot quando um jogador despausar o jogo.
AVERAGE LAG Takes a screenshot when any player's ping is average (every 5 seconds). Tira uma screeenshot quando o ping de qualquer jogador estiver médio (a cada 5 segundos).
HIGH LAG Takes a screenshot when any player's ping is high (every 5 seconds). Tira uma screeenshot quando o ping de qualquer jogador estiver alto (a cada 5 segundos).
KILL Takes a screenshot when you kill another player. Tira uma screeenshot quando você matar outro jogador.
DEATH Takes a screenshot when another player kills you. Tira uma screeenshot quando outro jogador matar você.
OBITUARY Takes a screenshot when any player dies. Tira uma screeenshot quando qualquer jogador morrer.
FLAG TAKEN Takes a screenshot when a flag is taken. Tira uma screeenshot quando uma bandeira for pega.
FLAG DROPPED Takes a screenshot when a flag is dropped. Tira uma screeenshot quando uma bandeira for derrubada.
FLAG RETURNED Takes a screenshot when a flag is returned. Tira uma screeenshot quando uma bandeira for retornada.
FLAG DELIVERED Takes a screenshot when a flag is delivered. Tira uma screeenshot quando uma bandeira for entregue.
GAMEOVER Takes a screenshot when a player runs out of lives. Tira uma screeenshot quando um jogador ficar sem vidas.
TERMINATOR Takes a screenshot when a player becomes the Terminator. Tira uma screeenshot quando um jogador virar o Terminator.
