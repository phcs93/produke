Now you can toggle the automatic weapon switch as well as set the weapon preference through the menu 
Options > Player Setup > Weapon Preference
. Agora você pode setar a troca automatica de arma como também a ordem de preferência das armas através do menu 
Options > Player Setup > Weapon Preference
. The automatic weapon switch (for when picking a new weapon) now has 3 modes: A troca automatica de armas (para quando uma nova arma é pega) agora possui 3 modos:
MODE MODO DESCRIPTION DESCRIÇÃO
OFF When picking a new weapon, the game will NEVER automatically switch to the new one. Ao pegar uma nova arma, o jogo NUNCA trocará automaticamente para a arma nova.
ON In this mode, the game will ALWAYS switch weapons when the weapon you picked is a new one. Nesse modo, o jogo SEMPRE trocará de arma quando para a arma que você pegou é uma arma nova.
BY PREFERENCE In this mode, the automatic weapn switch will only occur in case the priority of the current weapon is inferior to the new one. Nesse modo, só ocorrerá a troca automatica ao pegar uma nova arma caso a prioridade da arma atual seja inferior á prioridade da arma nova.
