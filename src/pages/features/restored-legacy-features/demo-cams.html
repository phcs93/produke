The game will use SE 27 (Demo Cam) to shift current player view when watching demos. You can enable this in the new menu 
Options > Recording Setup > Playback Options
. O jogo vai utilizar o SE 27 (Demo Cam) para trocar a visão do player atual quando estiver assistindo demos. Você pode habilitar essa funcionalidade no novo menu
