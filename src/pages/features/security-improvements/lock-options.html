You can start the game with the argument 
/lockoptions
to prevent other players from changing the netplay options. Only one player can use this argument, otherwise the game won't start. Você pode iniciar o jogo com o argumento 
/lockoptions
para impedir que os demais jogadores modifiquem as netplay options. Apenas um jogador pode usar esse argumento, caso contrário o jogo não iniciará.
