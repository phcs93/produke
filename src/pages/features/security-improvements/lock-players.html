You can start the game with the argument 
/lockplayers
to prevent other players from changing their name, color and team. Only one player can use this argument, otherwise the game won't start. Você pode iniciar o jogo com o argumento 
/lockplayers
para impedir que os demais jogadores troquem de nome, cor e time. Apenas um jogador pode usar esse argumento, caso contrário o jogo não iniciará.
