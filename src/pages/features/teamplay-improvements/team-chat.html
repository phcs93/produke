You can send a message to only your team or to a specific player. Você pode mandar uma mensagem para apenas o seu time ou para um jogador especifico.
You can use a few hotkeys to speed up chat: Você pode utilizar alguns atalhos para agilizar o chat:
HOTKEY ATALHO DESCRIPTION DESCRIÇÃO
* Send the message to everyone. Example:
Typing *proDuke is the future will send the message "proDuke is the future" to everyone. Envia a mensagem para todos. Exemplo:
Digitar *proDuke is the future enviará a mensagem "proDuke is the future" para todos.
# Sends the message only to your team. Example:
Typing #CTF is amazing! will send the message "CTF is amazing!" to your team allies. Envia a mensagem apenas para o seu time. Exemplo:
Digitar #CTF is amazing! enviará a mensagem "CTF is amazing!" para todos do seu time.
@P Send the message only to player P, where P is the player number. Example:
Typing @3You suck! will send the message "You suck!" to the player 3 (the player order is the same on the score bars, counting from 1 onwards). Envia a mensagem apenas para o jogar P, onde P é o número do jogador. Exemplo:
Digitar @3You suck! enviará a mensagem "You suck!" para o player 3 (a ordem dos players é a mesma da barra de pontuação, com a contagem começando em 1).
You can change the chat behavior to 3 differente modes in the menu 
Options > Game Options > Chat Mode
. Você pode mudar o comportamento do chat para 3 modos diferentes no menu 
Options > Game Options > Chat Mode
.
MODE MODO DESCRIPTION DESCRIÇÃO
CLASSIC In this mode, before the message is sent, the game asks you who you want to send the message to (like in the screenshot above), unless the message hotkeys are used. Neste modo, antes que a mensagem seja enviada, o jogo pergunta para quem você quer enviar a mensagem (como na screenshot acima), a não ser que os atalhos de mensagens sejam usados.
BROADCAST In this mode, the game doesn't ask who you want to send the message to, it just sends it to everyone. If you want to send a private message to a player or your team then you'll have to use chat hotkeys. Neste modo, o jogo não pergunta para quem você quer enviar a mensagem, ela será enviada para todos. Se você quiser enviar uma mensagem privada para um jogador ou para o seu time você precisará utilizar os atalhos de mensagem.
TO TEAM In this mode, the game doesn't ask who you want to send the message to, it will send the message to your team if playing a team mode, otherwise it just sends it to everyone. If you want to send a private message to a player or to everyone then you'll have to use chat hotkeys, like explained previously. Neste modo, o jogo não pergunta para quem você quer enviar a mensagem, a mensagem será enviada para o seu time caso estiver jogando um modo de time, caso contrário, será enviado enviada para todos. Se você quiser enviar uma mensagem privada para um jogador ou para todos você precisará utilizar os atalhos de mensagem, como explicado anteriormente.
