The original Friendly Fire option is now extended to team games (i.e. if toggled off, you will not be able to hurt your teammates). This feature can be enabled at the menu 
Netplay Options > NetFlags > [COOP/TEAM] Friendly Fire
. Note that proDuke fixes the bug that made it impossible to kill yourself in case Friendly Fire was disabled, but that behavior can be restored via the netflag 
Netplay Options > NetFlags > [DOS] No Suicide If FFire Is Off
. A opção original de Fogo Amigo foi extendida para os modos de time (i.e. se desabilitada, você não infringirá dano em seus aliados). Este recurso pode ser habilitado no menu 
Netplay Options > NetFlags > [COOP/TEAM] Friendly Fire
. Note que no proDuke o bug original de não poder se matar quando o fogo amigo estiver desabiltiado foi arrumado, mas esse comportamento pode ser restaurado pela netflag 
Netplay Options > NetFlags > [DOS] No Suicide If FFire Is Off
.
