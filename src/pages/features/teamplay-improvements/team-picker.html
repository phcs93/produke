For team based modes, a team picker is shown at the terminal. After each player has selected a team, the game will start. The game needs to be started with the 
/teampicker
command line. Para modos baseados em times, um seletor de times é exibido no terminal. Depois que cada jogador selecionar um time, o jogo é iniciado. O jogo precisa ser iniciado com a linha de comando 
/teampicker
.
