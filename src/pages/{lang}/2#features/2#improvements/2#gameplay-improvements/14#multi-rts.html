Support for Multiple RTS files loaded. Each player can have it's own distinct RTS file selected, and for the sound playback the game will load the RTS file by demand. The end result is that everyone will be able to hear the corresponding sounds of the RTS files you choose, even if their curently selected RTS is diferent from yours! Yes, this means that it's possible to, for example, have a sound from DUKE.RTS and SW.RTS playing at the same time! But, of course, all players need to have matching .rts files on the /rts folder. You can select your RTS file through 
OPTIONS > PLAYER OPTIONS > REMOTE RIDICULE
. The F8 key will also be a shortcut to that. Or, you can use the new "RTS" console command. Make sure to enable the RTS sound playback by setting 
OPTIONS > SOUND SETUP > VOICE QUOTES
to FULL. Suporte para Multiplos RTS carregados. Cada jogador poderá ter selecionado um arquivo RTS distinto e na hora de tocar os sons de um RTS o jogo o carregará sob demanda. O resultado final é que todos serão capazes de escutar os sons do seu RTS, mesmo que os outros estejam com outra RTS selecionada! Sim, isso significa que será possível, por exemplo, ter um som do DUKE.RTS tocando ao mesmo tempo em que um som do SW.RTS também esteja! Porém, obviamente é necessário que todos compartilhem os mesmos arquivos RTS em suas pastas /rts. Você poderá selecionar sua RTS pelo menu 
OPTIONS > PLAYER OPTIONS > REMOTE RIDICULE
. A tecla F8 servirá de atalho para isso. Ou, você também poderá utilizar o comando "RTS" no console. Certifique-se de habilitar os sons RTS trocando a opção em 
OPTIONS > SOUND SETUP > VOICE QUOTES
para FULL. Holding the ALT key now shows the sound names of the currently selected RTS file. This can be disabled on the menu 
OPTIONS > HUD/MENU SETUP > RTS SOUND NAMES
. Ao segurar a tecla ALT os nomes dos sons do RTS selecionado serão exibidos. Isso pode ser desabilitado pelo menu 
OPTIONS > HUD/MENU SETUP > RTS SOUND NAMES
. Check the configuration guide. Confira o guia de configuração.
