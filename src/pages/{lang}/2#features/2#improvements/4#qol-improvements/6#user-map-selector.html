You can now load an user map directly from the game's menu. In between "Episode 4" and "Episode 1" there will be a new option called 
User Map
. Você pode carregar um user map diretamente pelo menu do jogo. Entre o "Episode 4" e o "Episode 1" terá uma nova opção chamada 
User Map
. Clicking on this new option will show a list of all maps in your configured user maps folder. You can press a character key and the cursor will be positioned at the first entry that begins with that letter. Clicar nessa nova opção vai exibir uma lista com todos os user maps que estiverem no diretório de user maps configurado. Apertar uma tecla fará com que o cursor seja posicionado no primeiro mapa que comece com a letra pressionada. The usermap listing screen can be filtered as well, like the DMO and RTS screens. A tela com a listagem de usermaps pode ser filtrada, assim como as telas de DMO e RTS. You can customize where the game looks for usermaps, check the configuration guide to learn how to set it up. Você pode customizar a pasta em que o jogo vai procurar os mapas, confira o guia de configuração para aprender como fazer.
