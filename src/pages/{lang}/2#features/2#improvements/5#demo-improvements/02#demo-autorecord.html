You can configure the game to always record demos through the new menu 
Options > Recording Setup > Record
. É possivel configurar o jogo para sempre gravar demos através do novo menu 
Options > Recording Setup > Record
. There are 4 options: Existem 4 opções:
OPTION OPÇÃO DESCRIPTION DESCRIÇÃO
OFF Never record demos. Nunca gravar demos.
ONCE Records only one match and then goes back to OFF. Gravar apenas uma partida e depois voltar para OFF.
ALWAYS Always records all matches. Sempre gravar todas as partidas.
NETPLAY ONLY Records only online matches. Gravar apenas partidas online.
