All of proDuke's new features and modifications are toggled via NetFlags. These are a collection of options that can be toggled in-game through the menu or on start-up through command line arguments. Todas as novas funcionalidades e modificações do proDuke podem ser ativadas/desativadas através das NetFlags. Elas são uma coleção de opções que podem ser alternadas dentro do jogo pelo menu ou na inicialização por argumentos na linha de comando.
