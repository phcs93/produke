The game now also uses Duke Nukem 3D World Tour weapon model sprites at wide-screen resolutions, so the weapons can be properly aligned at the middle of the screen without showing the cut edges. You need to place the Duke Nukem 3D World Tour ART files in the game's directory in order for it to work. You can obtain these ART files from your original copy of Duke Nukem 3D World Tour. If you got it from Steam, right click the game and select 
Manage > Explore Local Files
. There you can find the 
TILES020.ART
, 
TILES021.ART
and 
TILES022.ART
ART files. Just copy these files to proDuke's game folder.
