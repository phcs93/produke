To configure your user maps directory, just edit your 
duke3d.cfg
file. Under the 
[Misc]
section there is a new entry called 
MAPDir
. You can set a full path or a relative path. It defaults to a 
"maps/"
in the root directory of the game executable. Para configurar o diretório de user maps, edite o seu arquivo 
duke3d.cfg
. Na categoria 
[Misc]
existirá uma nova entrada chamada 
MAPDir
. Você pode configurar um caminho completo ou um caminho relativo. Por padrão ele apontará para 
"maps/"
na pasta raiz do executável do jogo. Some examples of how to set it up: Alguns exemplos de como configurar: A "maps" folder in the directory of the game executable: Pasta "maps" no mesmo diretório do executável do jogo: 
[Misc]
MAPDir = "maps"
A "maps" folder in the parent directory of the game executable, using relative path: Pasta "maps" no diretório pai de onde está o executavel do jogo, utilizando caminho relativo: 
[Misc]
MAPDir = "../maps"
A "maps" folder in some absolute path: Pasta "maps" em um caminho absoluto: 
[Misc]
MAPDir = "C:\Games\Build Engine Games\Duke Nukem 3D\maps"
